# Projet React

Ce projet a été créé avec Create React App.

## Commandes principales

### Démarrer l'application

Pour lancer l'application en mode développement :
```
npm start
```

L'application s'ouvre dans le navigateur à l'adresse http://localhost:3000

La page se recharge automatiquement quand on modifie le code.

### Lancer les tests

Pour exécuter les tests :
```
npm test
```

### Créer la version de production

Pour préparer l'application pour la mise en ligne :
```
npm run build
```

Cette commande crée un dossier `build` avec les fichiers optimisés prêts à être déployés.
